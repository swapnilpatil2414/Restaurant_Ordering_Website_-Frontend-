# 🍽️ Restaurant Ordering Website (Frontend)

This is a **Restaurant Ordering Website** built using **HTML, CSS, and JavaScript**.  
It allows users to view a menu, add items to the cart, and see the order summary.

---

## 🚀 Features
- Menu items with **images, names, prices, and descriptions**  
- **Add to Cart** button for each item  
- Shopping cart shows **selected items, quantities, and total amount**  
- Responsive design for **mobile and desktop**  
- Clean and modern UI for a restaurant website  

---

## 🛠️ Tech Stack
- **HTML5** – Page structure  
- **CSS3** – Styling & layout  
- **JavaScript** – Cart functionality
